1. Introducción

Este documento describe la API del servicio Workers Service, que proporciona endpoints para gestionar datos de trabajadores, archivos de trabajadores, y carga masiva de datos mediante archivos Excel. Conforme a la especificación Swagger 2.0.

2. Información General

  • Versión: 0.0.23

  • Contacto: Angel Diaz (Angel_Diaz_Sanchez@wvi.org)

  • Servidor: https://prod.svc-workers.ecms.setrass.net

3.1.2. Aprobar Trabajador

Aprueba un trabajador existente por su ID.

Path: /inspection/workers/v1/approved-worker/{workerId}

Method: PATCH

Parameters: - workerId (path, string, required): ID del trabajador (UUID)

Responses: - 200 OK:

{
  "errorCode": 0,
  "statusError": "SUCCESS",
  "errorMessages": []
}

3.1.3. Verificar Existencia de Trabajador

Verifica si un trabajador existe por su ID.

Path: /inspection/workers/v1/check-ExistenceStaff/{id}

Method: GET

Parameters: - id (path, string, required): ID del trabajador (UUID)

Responses: - 200 OK:

true

3.1.13. Actualizar ID de Usuario

Actualiza el ID de usuario asociado a un trabajador.

Path: /inspection/workers/v1/update-user-id/{worker_id}/{new_user_id}

Method: PATCH

Parameters: - worker_id (path, string, required): ID del trabajador (UUID) - new_user_id (path, string, required): Nuevo ID de usuario (UUID)

Responses: - 200 OK:

{
  "errorCode": 0,
  "statusError": "SUCCESS",
  "errorMessages": []
}

3.2. Worker Files Controller

Agrega una lista de archivos asociados a un trabajador.

Path: /inspection/worker-files/v1/add-new-list-worker-files

Method: POST

Parameters: - applicantFilesDtoList (body, array[WorkerFilesDto], required): Lista de archivos

Responses: - 200 OK:

Introducción

Este documento describe la API del Servicio de Personal, diseñada para gestionar información de empleados, roles, asignaciones, auditorías y archivos relacionados en el sistema SETRASS. La API permite crear, actualizar, buscar y eliminar registros de personal, así como manejar referencias laborales y archivos asociados.

Descripción General

La API está organizada en los siguientes controladores:

  • Controlador de Auditorías: Gestiona registros de auditoría del personal.

  • Controlador de Errores Básico: Maneja respuestas de error.

  • Controlador de Puestos: Administra información de puestos laborales.

  • Controlador de Archivos de Referencia: Gestiona archivos de referencia laboral.

  • Controlador de Personal: Maneja registros de empleados y sus asignaciones.

  • Controlador de Archivos de Personal: Administra archivos asociados a empleados.

Todos los puntos finales son accesibles a través del host prod.svc-staff.ecms.setrass.net/.

Controlador de Auditorías

Este controlador gestiona registros de auditoría relacionados con el personal.

Listar Auditorías de Personal

Recupera una lista paginada de auditorías de personal.

Método HTTP: GET Ruta: /staff/audits/v1/listStaffAudits/{page}/{size} Resumen: Lista auditorías de personal con paginación. Produce: / Parámetros: - page (Ruta, Requerido): Número de página (entero). - size (Ruta, Requerido): Tamaño de página (entero).

Ejemplo de Solicitud:

GET /staff/audits/v1/listStaffAudits/0/10

Definiciones

La API utiliza los siguientes modelos de datos, definidos en el esquema Swagger:

  • StaffGeneralInfoDto: Representa un empleado con detalles personales y laborales.

  • JobGeneralDto: Modelo para información de puestos laborales.

  • StaffFilesDto y ReferenceFilesDto: Modelos para gestionar archivos asociados a empleados.

  • CreateUserGeneralInfoDto: Estructura para información de usuario.

  • ResponseAdvisor: Estructura común para respuestas con información de estado.

  • AuditDto: Modelo para registros de auditoría.

Para definiciones detalladas del esquema, consulte la especificación JSON de Swagger en prod.svc-staff.ecms.setrass.net/swagger/v1/swagger.json.

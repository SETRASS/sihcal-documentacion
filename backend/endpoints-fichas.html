1. Introducción

Este documento describe la API del módulo de Inspección (FICHAS Inspection-Service). Proporciona endpoints para gestionar datos de trabajadores, empleadores, registros, fichas, y más, conforme a la especificación OpenAPI 3.0.1.

2. Información General

  • Versión: 0.0.350

  • Contacto: Marvin Toro (marvin_toro@wvi.org)

  • Licencia: Apache 2.0 (http://www.apache.org/licenses/LICENSE-2.0.html)

  • Servidor: https://prod.svc-inspection.ecms.setrass.net

3. Endpoints

3.1. Worker Controller

3.1.1. Añadir Nuevo Trabajador

Crea un nuevo trabajador.

Path: /inspection/sheets-workerData/v1/add-new-worker

Method: POST

Request Body:

{
  "workerId": "123e4567-e89b-12d3-a456-426614174000",
  "name": "Juan Pérez",
  "dni": "12345678A",
  "status": "ACTIVO"
}

Responses: - 200 OK:

{
  "workerId": "123e4567-e89b-12d3-a456-426614174000",
  "name": "Juan Pérez",
  "dni": "12345678A",
  "status": "ACTIVO",
  "responseAdvisor": {
    "code": "200",
    "message": "Trabajador creado exitosamente",
    "timestamp": "2025-05-06T10:00:00Z",
    "path": "/inspection/sheets-workerData/v1/add-new-worker",
    "method": "POST"
  }
}

1. Introducción

Este documento describe la API del servicio Lookups Service, que proporciona endpoints para gestionar datos de oficinas, ubicaciones, y datos generales de personal. Conforme a la especificación Swagger 2.0.

2. Información General

  • Versión: 0.0.43

  • Contacto: Angel Diaz Sanchez (angel_diaz_sanchez@wvi.org)

  • Servidor: https://prod.svc-lookups.ecms.setrass.net

3. Endpoints

3.1. Offices Controller V1

3.1.1. Obtener Todas las Oficinas

Obtiene todas las oficinas disponibles.

Path: /Offices/v1/all

Method: GET

Responses: - 200 OK:

[
  {
    "id": "123e4567-e89b-12d3-a456-426614174000",
    "isoNumber": "ISO123",
    "office": "Oficina Central",
    "officeType": "PRINCIPAL",
    "parentId": null
  }
]

3.1.2. Obtener Oficinas por ID Padre

Obtiene todas las oficinas asociadas a un ID padre.

Path: /Offices/v1/office/{IdParent}

Method: GET

Parameters: - IdParent (path, string, required): ID del padre

Responses: - 200 OK:

[
  {
    "id": "223e4567-e89b-12d3-a456-426614174001",
    "isoNumber": "ISO124",
    "office": "Oficina Secundaria",
    "officeType": "SECUNDARIA",
    "parentId": "123e4567-e89b-12d3-a456-426614174000"
  }
]

3.1.3. Obtener Oficina por Número ISO

Obtiene una oficina por su número ISO.

Path: /Offices/v1/officeByIsoNumber/{IsoNumber}

Method: GET

Parameters: - IsoNumber (path, string, required): Número ISO de la oficina

Responses: - 200 OK:

{
  "id": "123e4567-e89b-12d3-a456-426614174000",
  "isoNumber": "ISO123",
  "office": "Oficina Central",
  "officeType": "PRINCIPAL",
  "parentId": null
}

3.1.4. Obtener Oficina por Nombre y Tipo

Obtiene una oficina por su nombre y tipo.

Path: /Offices/v1/officeByNameAndType/{officeName}/{officeType}

Method: GET

Parameters: - officeName (path, string, required): Nombre de la oficina - officeType (path, string, required): Tipo de oficina

Responses: - 200 OK:

{
  "id": "123e4567-e89b-12d3-a456-426614174000",
  "isoNumber": "ISO123",
  "office": "Oficina Central",
  "officeType": "PRINCIPAL",
  "parentId": null
}

3.1.5. Obtener Jerarquía de Oficinas

Obtiene la jerarquía de oficinas según filtros.

Path: /Offices/v1/officeHierarchy/{filters}

Method: GET

Parameters: - filters (path, string, required): Filtros para la jerarquía

Responses: - 200 OK:

[
  {
    "office": {
      "id": "123e4567-e89b-12d3-a456-426614174000",
      "isoNumber": "ISO123",
      "office": "Oficina Central",
      "officeType": "PRINCIPAL",
      "parentId": null
    },
    "childrenOffice": []
  }
]

    },
    "children": []
  }

3.3.2. Obtener Departamento y Municipio por Código

{
  "departamento": {
    "id": "123e4567-e89b-12d3-a456-426614174014",
    "code": "DEP001",
    "isoName": "ISO_DEP001",

    "locationType": "DEPARTMENT",
    "parentId": null
  },
  "municipio": {

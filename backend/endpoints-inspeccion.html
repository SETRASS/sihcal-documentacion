1. Introducción

Este documento describe la API del módulo de Inspección (Inspection-Service). Proporciona endpoints para gestionar datos de trabajadores, empleadores, registros, citas, y más, conforme a la especificación OpenAPI 3.0.1.

  • Versión: 0.0.350

3. Endpoints

3.1. Records Controller

3.1.1. Añadir Nuevo Registro

Crea un nuevo registro.

Path: /inspection/records/v1/add-new

Method: POST

Request Body:

{
  "typeRequest": "123e4567-e89b-12d3-a456-426614174000",
  "regionalOffice": "Madrid",
  "localOffice": "Madrid Central",
  "objective": "987fcdeb-12ab-34cd-5678-426614174000",
  "recordDescription": "Inspección laboral",
  "staffId": "456e7890-e89b-12d3-a456-426614174000",
  "causes": [
    {
      "id": "789fcdeb-12ab-34cd-5678-426614174000",
      "name": "Incumplimiento normativo"
    }
  ],
  "departmentCode": "MAD",
  "municipalityCode": "MAD01"
}

Responses: - 200 OK:

{
  "recordId": "123e4567-e89b-12d3-a456-426614174001",

    "code": "200",

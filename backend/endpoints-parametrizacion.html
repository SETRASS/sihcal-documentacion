1. Introducción

Este documento describe la API del servicio Assignments Service, que proporciona endpoints para gestionar asignaciones de inspecciones y conciliaciones, así como auditorías relacionadas. Conforme a la especificación Swagger 2.0.

2. Información General

  • Versión: 1.0.0.34

  • Contacto: Marvin Toro (marvin_toro@wvi.org.com)

  • Servidor: https://prod.svc-assignment.ecms.setrass.net

3. Endpoints

3.1. Assignment Controller

3.1.1. Agregar Nueva Asignación

Agrega una nueva asignación al sistema.

Path: /assignment-data/v1/add-new-assignment

Method: POST

Parameters: - assignmentDto (body, AssignmentDto, required): Datos de la asignación

Responses: - 200 OK:

{
  "errorCode": 0,
  "statusError": "SUCCESS",
  "errorMessages": []
}

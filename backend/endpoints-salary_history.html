1. Introducción

Este documento describe la API del servicio Salary History Catalog Service, que proporciona endpoints para gestionar datos relacionados con el historial salarial, catálogos de actividades económicas, cantidades de empleados por rangos y bonos educativos. Conforme a la especificación Swagger 2.0.

2. Información General

  • Versión: 0.0.19

  • Contacto: Angel Darid Diaz (angel_diaz_sanchez@wvi.org)

  • Servidor: https://prod.svc-salary-history.ecms.setrass.net

3. Endpoints

3.1. EA Catalog Controller V1

3.1.1. Obtener Todos los Catálogos de Actividades Económicas

Obtiene todos los catálogos de actividades económicas disponibles.

Path: /salary-history/ea-catalog/v1/all

Method: GET

Responses: - 200 OK:

[
  {
    "id": "123e4567-e89b-12d3-a456-426614174000",
    "name": "Industria Manufacturera",
    "description": "Actividades relacionadas con la producción de bienes",
    "active": true,
    "versionTag": "1.0"
  }
]

3.1.2. Obtener Catálogos de Actividades Económicas por Nombre

Obtiene catálogos de actividades económicas filtrados por nombre.

Path: /salary-history/ea-catalog/v1/getAllByName/{name}

Method: GET

Parameters: - name (path, string, required): Nombre de la actividad económica

Responses: - 200 OK:

[
  {
    "id": "123e4567-e89b-12d3-a456-426614174000",
    "name": "Industria Manufacturera",
    "description": "Actividades relacionadas con la producción de bienes",
    "active": true,
    "versionTag": "1.0"
  }
]

3.2. Employee Qty Waves Controller V1

3.2.1. Obtener Todos los Rangos de Cantidad de Empleados

Obtiene todos los rangos de cantidad de empleados disponibles.

Path: /salary-history/employee-qty-waves/v1/all

Method: GET

Responses: - 200 OK:

[
  {
    "id": 1,
    "description": "Pequeña Empresa",
    "minQty": 1,
    "maxQty": 50
  },
  {
    "id": 2,
    "description": "Mediana Empresa",
    "minQty": 51,
    "maxQty": 250
  }
]

3.3. Salary Rate Per Year Controller V1

3.3.1. Obtener Tabla de Tasas Salariales por Año

Obtiene una tabla de tasas salariales por año.

Path: /salary-history/salary-rate-per-year/v1/getTable

Method: GET

Responses: - 200 OK:

[
  {
    "year": 2023,
    "salaryRates": [
      {
        "economicActivity": "Industria Manufacturera",
        "companyEmployeeWeight": "Pequeña",
        "perHour": 10.5,
        "perDay": 84.0,
        "perMonth": 1800.0
      }
    ]
  }
]

3.3.2. Obtener Salario Mínimo por Año y Actividad Económica

Obtiene el salario mínimo por año y actividad económica.

Path: /salary-history/salary-rate-per-year/v1/min-salary-per-years-and-ea/{eaId}/years/{years}

Method: GET

Parameters: - eaId (path, string, required): ID de la actividad económica - years (path, string, required): Año(s) a consultar

Responses: - 200 OK:

[
  {
    "economicActivityId": "123e4567-e89b-12d3-a456-426614174000",
    "economicActivityName": "Industria Manufacturera",
    "minSalary": 1800.0,
    "year": 2023
  }
]

3.3.3. Obtener Salario Mínimo por Año, Actividad Económica y Tamaño de Empresa

Obtiene el salario mínimo por año, actividad económica y tamaño de empresa.

Path: /salary-history/salary-rate-per-year/v1/min-salary-per-years-and-ea-and-companySize/{eaId}/years/{years}/companySize/{companySize}

Method: GET

Parameters: - eaId (path, string, required): ID de la actividad económica - years (path, string, required): Año(s) a consultar - companySize (path, integer, required): Tamaño de la empresa (número de empleados)

Responses: - 200 OK:

[
  {
    "economicActivityId": "123e4567-e89b-12d3-a456-426614174000",
    "economicActivityName": "Industria Manufacturera",
    "minSalary": 1800.0,
    "year": 2023
  }
]

3.4. Salary Rate Per Year Educational Bonus Controller V1

3.4.1. Obtener Tasas Salariales con Bono Educativo

Obtiene tasas salariales por año con bono educativo según actividad económica, años y cantidad.

Path: /salary-history/salary-rate-per-year-educational-bonus/v1/salary-per-years-educational-bonus/{eaId}/years/{years}/quantity/{quantity}

Method: GET

Parameters: - eaId (path, string, required): ID de la actividad económica - years (path, string, required): Año(s) a consultar - quantity (path, integer, required): Cantidad de empleados

Responses: - 200 OK:

[
  {
    "economicActivityId": "123e4567-e89b-12d3-a456-426614174000",
    "economicActivityName": "Industria Manufacturera",
    "minSalary": 1800.0,
    "amountEduc": 200.0,
    "year": 2023
  }
]
